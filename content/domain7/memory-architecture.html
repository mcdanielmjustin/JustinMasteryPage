Memory Architecture & Working Memory

Domain 7: Biopsychology ~25 min read
Flashcards Quick Quiz: Fundamental Advanced Learning

The Atkinson-Shiffrin Multi-Store Model

The Atkinson-Shiffrin model (1968) remains the foundational framework for understanding memory architecture. It proposes three distinct memory stores that differ in capacity, duration, and encoding format. Information flows sequentially from sensory memory to short-term memory and, with rehearsal, to long-term memory.

Sensory Memory

Sensory memory holds a brief, relatively complete record of sensory experience. Two forms are most studied:

  • Iconic memory (visual): Demonstrated by Sperling's (1960) partial-report paradigm, iconic memory holds a nearly complete visual image for approximately 250-500 milliseconds. Sperling showed that participants could report any row of a briefly flashed letter array when cued immediately, but this advantage disappeared after about half a second, indicating rapid decay of the iconic trace.
  • Echoic memory (auditory): Retains auditory information for approximately 3-4 seconds. This longer duration explains why you can "replay" something someone just said, even if you were not initially attending to it.
Sperling, G. (1960). The information available in brief visual presentations. Psychological Monographs: General and Applied, 74(11), 1-29.

Short-Term Memory (STM)

Short-term memory holds a limited amount of information in an active, accessible state for approximately 15-30 seconds without rehearsal. George Miller's (1956) classic paper established that STM capacity is approximately 7 plus or minus 2 items (the "magical number seven").

Chunking:

Chunking is the process of grouping individual items into larger meaningful units, effectively increasing the amount of information that can be held in short-term memory. For example, the sequence 1-9-4-1-1-7-7-6 contains 8 digits, but chunking it as 1941-1776 (two historical dates) reduces the memory load to 2 chunks. Expertise in a domain facilitates chunking — chess masters perceive meaningful board configurations as single chunks rather than individual piece positions.

Baddeley's Working Memory Model

Alan Baddeley and Graham Hitch (1974) proposed the working memory model as a replacement for the unitary short-term store in the Atkinson-Shiffrin model. Working memory is not merely a passive holding area but an active system that both maintains and manipulates information during cognitive tasks.

The Phonological Loop

The phonological loop consists of a phonological store (which holds speech-based information for 1-2 seconds) and an articulatory rehearsal process (which refreshes the contents through subvocal repetition). Two key phenomena demonstrate the phonological loop's operation:

  • Phonological similarity effect: Lists of similar-sounding items (B, D, G, T, P) are harder to recall than dissimilar items, because they produce confusable representations in the phonological store.
  • Word-length effect: Lists of long words are harder to recall than short words, because longer words take more time to rehearse in the articulatory loop.

The Visuospatial Sketchpad

The visuospatial sketchpad maintains and manipulates visual and spatial information. It can be subdivided into a visual cache (which stores visual patterns and colors) and an inner scribe (which processes spatial and movement information). Tasks such as mental rotation, navigating a familiar environment from memory, and solving jigsaw puzzles rely on this component.

EPPP Application:

The working memory model predicts that two tasks using the same subsystem will interfere with each other more than two tasks using different subsystems. A patient asked to remember a phone number (phonological loop) while counting backward (also phonological loop) will show greater interference than if asked to remember the phone number while tracking a moving object (visuospatial sketchpad). This principle of selective interference is a key prediction tested on the EPPP.

Baddeley, A. D. (2000). The episodic buffer: A new component of working memory? Trends in Cognitive Sciences, 4(11), 417-423.

Long-Term Memory Taxonomies

Long-term memory is not a unitary system but comprises multiple distinct systems that differ in their neural substrates, the type of information they store, and whether they require conscious awareness for retrieval.

Declarative vs. Procedural Memory

Declarative (explicit) memory refers to memories that can be consciously recalled and verbally declared. It is further divided into episodic memory (autobiographical events situated in time and place) and semantic memory (general world knowledge and facts). Declarative memory depends critically on the hippocampus and medial temporal lobe structures.

Procedural (implicit) memory refers to knowledge of how to perform skills and procedures (e.g., riding a bicycle, typing). Procedural memories are acquired gradually through practice and are expressed through performance rather than conscious recollection. This system depends on the basal ganglia, cerebellum, and motor cortex rather than the hippocampus.

Tulving's Distinction: Episodic vs. Semantic Memory

Endel Tulving (1972) proposed that episodic memory stores personally experienced events with contextual details (time, place, associated emotions) and involves "mental time travel." Semantic memory stores general knowledge, facts, concepts, and vocabulary, independent of the personal context in which they were learned. Knowing that Paris is the capital of France (semantic) is distinct from remembering your trip to Paris (episodic). This distinction is critical for understanding amnesia: patients with hippocampal damage may lose the ability to form new episodic memories while retaining the ability to acquire some new semantic knowledge.

Explicit vs. Implicit Memory

The distinction between explicit memory (conscious, intentional recollection) and implicit memory (unconscious influence of past experience on current behavior) is one of the most important in memory research and is heavily tested on the EPPP.

Priming

Priming is a form of implicit memory in which exposure to a stimulus influences the processing of a subsequent stimulus, without conscious awareness. After seeing the word "DOCTOR," a person is faster to recognize "NURSE" because the semantic network has been pre-activated. Priming can be perceptual (based on physical features) or conceptual (based on meaning).

Clinical Significance:

The explicit/implicit distinction has profound clinical implications. Amnesic patients who cannot consciously remember learning a new skill (explicit deficit) may nonetheless show normal skill improvement across practice sessions (intact implicit memory). Patient H.M. could learn new motor skills (mirror drawing) despite having no explicit memory of ever having practiced the task. This dissociation demonstrates that explicit and implicit memory rely on different neural systems.

Connectionist and Parallel Distributed Processing Models

In contrast to the discrete-store approach of Atkinson-Shiffrin, connectionist models (also called parallel distributed processing or PDP models) propose that memories are represented as patterns of activation across networks of interconnected processing units. Rather than being stored in a specific location, a memory emerges from the pattern of connection strengths across the network.

These models capture several important properties of human memory, including content-addressable retrieval (partial cues can activate complete memories), graceful degradation (the system continues to function despite partial damage), and the ability to generalize from specific experiences to novel situations. Connectionist models also provide a framework for understanding how memories can be distorted — if the network has encoded similar patterns, retrieval of one pattern may be contaminated by elements of another.

Exam Tip:

The EPPP may contrast the Atkinson-Shiffrin model (serial, stage-based) with the working memory model (parallel subsystems) and connectionist models (distributed representations). Know the key features and limitations of each: Atkinson-Shiffrin oversimplifies the flow of information; the working memory model better explains active processing but may overemphasize modularity; connectionist models capture neural plausibility but can be difficult to test empirically.

EPPP Exam Focus Points

  • Atkinson-Shiffrin model: sensory memory (brief, high capacity) → short-term memory (limited capacity, ~30 sec) → long-term memory (unlimited capacity, permanent)
  • Iconic memory (Sperling): visual sensory memory lasting ~250-500 ms; Echoic memory: auditory sensory memory lasting ~3-4 seconds
  • Miller's magical number: STM capacity is 7 plus or minus 2 items; chunking increases effective capacity
  • Baddeley's working memory: phonological loop (verbal), visuospatial sketchpad (visual-spatial), central executive (attention control), episodic buffer (integration)
  • Selective interference: tasks sharing the same working memory subsystem interfere more than tasks using different subsystems
  • Declarative memory (explicit, hippocampus-dependent) vs. procedural memory (implicit, basal ganglia/cerebellum-dependent)
  • Episodic memory (personal events in time/place) vs. semantic memory (general knowledge, facts) — both are declarative
  • Priming: a form of implicit memory; amnesic patients show intact priming despite impaired explicit memory
  • Amnesic patients demonstrate the explicit/implicit dissociation: impaired conscious recollection but intact procedural learning and priming
  • The central executive is the most clinically relevant working memory component; deficits are associated with prefrontal cortex damage and ADHD
