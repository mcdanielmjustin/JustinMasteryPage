Memory & Forgetting

Encoding, storage, retrieval, amnesia types

Overview of Memory

Memory is the cognitive process by which information is encoded, stored, and retrieved. Understanding memory is essential for the EPPP because memory processes underlie learning, clinical assessment, neuropsychological evaluation, and therapeutic interventions. Disruptions to memory systems are hallmark features of numerous neurological and psychological disorders.

Memory: The persistence of learning over time through the encoding, storage, and retrieval of information. Memory is not a unitary system but rather a collection of interrelated processes supported by distinct neural substrates.

Encoding: Getting Information In

Encoding is the initial processing of information that leads to a mental representation in memory. Not all information that enters the sensory system is encoded into long-term storage; the manner and depth with which we process information significantly determines what is remembered.

Levels of Processing (Craik & Lockhart, 1972)

The levels-of-processing framework proposes that the depth at which information is processed during encoding determines its durability in memory. Deeper processing leads to more elaborate, longer-lasting, and stronger memory traces.

  • Structural (shallow) processing: Encoding based on physical features of the stimulus, such as the appearance of printed words (e.g., "Is this word in uppercase letters?"). This produces the weakest memory traces.
  • Phonemic (intermediate) processing: Encoding based on how a stimulus sounds (e.g., "Does this word rhyme with 'train'?"). This produces moderate memory strength.
  • Semantic (deep) processing: Encoding based on the meaning of information (e.g., "Does this word fit in the sentence: 'The _____ crossed the road'?"). This produces the strongest and most durable memories.

Other Encoding Phenomena

Self-Reference Effect

Information related to the self is processed more deeply and remembered better than information about others. Asking "Does this word describe me?" leads to superior recall compared to semantic processing alone (Rogers, Kuiper, & Kirker, 1977).

Elaborative Encoding

The process of linking new information to existing knowledge and forming rich associations. The more connections created, the more retrieval routes are available later.

Encoding Specificity Principle (Tulving & Thomson, 1973)

Memory is most effective when the conditions at retrieval match the conditions present at encoding. This principle underlies both context-dependent memory (physical environment) and state-dependent memory (internal physiological or emotional state).

EPPP Application: A student who studies in a quiet room will recall material better when tested in a similar quiet environment. Similarly, material learned while in a particular mood is best recalled in that same mood (Bower, 1981).

Storage: Retaining Information

Storage refers to maintaining encoded information over time. The dominant model of memory storage distinguishes three systems based on duration and capacity.

Sensory Memory

The brief retention of sensory information. Iconic memory (visual) lasts approximately 0.5 seconds and was demonstrated by Sperling's (1960) partial-report procedure. Echoic memory (auditory) persists for 3-4 seconds.

Short-Term Memory (STM)

A limited-capacity store that holds information for brief periods. George Miller (1956) established the capacity of STM at 7 plus or minus 2 items. Capacity can be expanded through chunking, the process of grouping individual items into larger meaningful units.

Long-Term Memory (LTM)

The relatively permanent and essentially limitless storage system. Transfer from STM to LTM is facilitated by rehearsal, elaboration, and meaningful processing. LTM is divided into two major subsystems:

  • Explicit (declarative) memory: Conscious recollection of facts and events
    • Episodic memory: Personal experiences and events (Tulving, 1972)
    • Semantic memory: General knowledge and facts about the world
  • Implicit (nondeclarative) memory: Memory without conscious awareness
    • Procedural memory: Motor skills and habits (e.g., riding a bicycle)
    • Priming: Facilitated processing of a stimulus due to prior exposure
    • Classical conditioning: Learned associations between stimuli

Working Memory (Baddeley & Hitch, 1974)

Baddeley's working memory model replaced the unitary concept of STM with a multicomponent system that actively manipulates information rather than passively holding it:

  • Central executive: An attentional control system that directs focus, coordinates the subsystems, and manages cognitive resources. It has limited capacity and is modality-free.
  • Phonological loop: Maintains and rehearses verbal and acoustic information. Consists of a phonological store (the "inner ear") and an articulatory rehearsal process (the "inner voice"). Explains the word-length effect (shorter words are recalled better because they can be rehearsed faster).
  • Visuospatial sketchpad: Maintains and manipulates visual and spatial information (the "inner eye"). Used in navigation, mental imagery, and spatial reasoning.
  • Episodic buffer (added 2000): Integrates information from the phonological loop, visuospatial sketchpad, and long-term memory into unified, coherent episodes. Serves as a limited-capacity temporary store.

Clinical Significance: Working memory deficits are observed in ADHD, schizophrenia, traumatic brain injury, and age-related cognitive decline. Neuroimaging consistently implicates the dorsolateral prefrontal cortex in working memory tasks.

Retrieval: Getting Information Out

Retrieval is the process of accessing stored information. The success of retrieval depends on the quality of encoding, the availability of retrieval cues, and the match between encoding and retrieval conditions.

Types of Retrieval

  • Recall: Retrieving information without explicit cues. More demanding than recognition. Free recall (produce items in any order), serial recall (produce items in order), and cued recall (use a hint) are subtypes.
  • Recognition: Identifying previously encountered information from a set of options. Generally easier than recall because the target item serves as its own retrieval cue.
  • Relearning (savings method): Ebbinghaus's measure showing that previously learned material is relearned faster, even when conscious recall fails. The difference in learning trials is the "savings score."

Retrieval Cues and Context

Effective retrieval depends on the availability and quality of retrieval cues. Tulving's encoding specificity principle states that a cue is effective only if it was encoded with the target information at study.

Example: Godden and Baddeley (1975) demonstrated context-dependent memory by having divers learn words either underwater or on land. Recall was significantly better when the retrieval environment matched the learning environment.

Forgetting

Forgetting is the inability to retrieve previously stored information. Several theories account for why forgetting occurs.

Ebbinghaus's Forgetting Curve (1885)

Hermann Ebbinghaus demonstrated that forgetting is most rapid shortly after learning and then levels off. Using nonsense syllables, he showed that approximately 50% of learned material is forgotten within one hour, and about 70% within 24 hours, with the rate of forgetting then declining substantially.

Interference Theory

Interference theory holds that forgetting occurs because other information competes with or disrupts the target memory:

  • Proactive interference: Previously learned information interferes with the ability to learn or recall new information. ("Old" disrupts "new.")
  • Retroactive interference: Newly learned information interferes with the ability to recall previously learned information. ("New" disrupts "old.")

EPPP Example: A student studies Domain 6 material, then studies Domain 7. If the Domain 7 material makes it harder to recall Domain 6 content, that is retroactive interference. If the previously studied Domain 6 content makes Domain 7 harder to learn, that is proactive interference.

Decay Theory

Proposes that memory traces weaken over time if not accessed or rehearsed. While decay likely contributes to forgetting in sensory and short-term memory, evidence suggests that interference rather than mere time passage is the primary cause of long-term forgetting.

Retrieval Failure

Information is available in memory but not accessible due to the absence of appropriate retrieval cues. The tip-of-the-tongue (TOT) phenomenon illustrates this: the person knows they know the information but temporarily cannot access it. TOT states increase with age.

Motivated Forgetting

Freud proposed that threatening or anxiety-provoking memories may be pushed out of conscious awareness through repression (unconscious) or suppression (conscious). While controversial, Anderson and Green (2001) demonstrated "think/no-think" suppression effects in laboratory settings.

Amnesia

Amnesia refers to severe memory loss that exceeds normal forgetting. Understanding amnesia types is critical for neuropsychological assessment on the EPPP.

Anterograde Amnesia

The inability to form new memories after the onset of amnesia. Patients cannot transfer new information from short-term to long-term storage, yet procedural memory and priming are often preserved.

Case of H.M. (Henry Molaison): Following bilateral medial temporal lobe resection (including the hippocampus) for epilepsy treatment, H.M. developed profound anterograde amnesia. He could not form new declarative memories but retained procedural learning ability, demonstrating the dissociation between explicit and implicit memory systems (Scoville & Milner, 1957).

Retrograde Amnesia

The inability to recall memories formed before the onset of amnesia. Retrograde amnesia typically follows Ribot's law (temporal gradient): more recent memories are more vulnerable to disruption than remote memories, because older memories have been more fully consolidated.

Other Amnesia Types

  • Transient global amnesia: Temporary anterograde (and sometimes retrograde) amnesia lasting hours, often in older adults. Cause is unclear but may involve transient ischemia.
  • Korsakoff's syndrome: Amnesia resulting from thiamine (vitamin B1) deficiency, typically due to chronic alcoholism. Characterized by severe anterograde amnesia, confabulation, and damage to the mammillary bodies and dorsomedial thalamus.
  • Dissociative (psychogenic) amnesia: Memory loss for personal information following psychological trauma, without identifiable neurological damage.

Clinical Significance: Korsakoff's syndrome patients produce confabulations -- fabricated memories that fill gaps in recall -- without intent to deceive. This distinguishes the condition from malingering and factitious disorders.

Neural Substrates of Memory

Memory is not localized to a single brain region but depends on distributed neural networks:

  • Hippocampus: Critical for consolidation of new declarative memories from short-term to long-term storage. Damage causes anterograde amnesia for declarative information.
  • Amygdala: Modulates emotional memory consolidation. Emotionally arousing events are remembered better partly through amygdala-hippocampal interactions.
  • Prefrontal cortex: Involved in working memory, strategic encoding, and retrieval monitoring.
  • Cerebellum and basal ganglia: Essential for procedural memory and motor skill learning.
  • Perirhinal and entorhinal cortices: Contribute to recognition memory and serve as relays between neocortex and hippocampus.

Memory Distortions and Constructive Memory

Memory is a reconstructive process, not a perfect recording. Elizabeth Loftus's research on the misinformation effect demonstrated that post-event information can alter memory for the original event (Loftus & Palmer, 1974). Leading questions (e.g., "How fast were the cars going when they smashed into each other?" vs. "...hit each other?") systematically influenced speed estimates and even whether participants "remembered" seeing broken glass.

  • Source monitoring errors: Misattributing the source of a memory (e.g., confusing something imagined with something experienced).
  • False memories: Deese-Roediger-McDermott (DRM) paradigm demonstrates that presenting lists of semantically related words reliably produces false memories of non-presented critical lure words.
  • Flashbulb memories: Vivid, confident memories for emotionally significant events. Despite high subjective confidence, accuracy of these memories declines over time similarly to ordinary memories (Talarico & Rubin, 2003).

Clinical Significance: Understanding memory distortion is essential for evaluating the reliability of eyewitness testimony, recovered memories, and children's testimony in forensic contexts. Clinicians must be aware that therapeutic techniques (e.g., guided imagery, hypnosis) can inadvertently create false memories.
