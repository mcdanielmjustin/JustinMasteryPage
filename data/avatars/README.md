# Avatar GLB Files

Place Ready Player Me `.glb` avatar files here.

## Required files:
- `adult_male.glb`
- `adult_female.glb`
- `young_male.glb`
- `young_female.glb`
- `adolescent_male.glb`
- `adolescent_female.glb`
- `elder_male.glb`
- `elder_female.glb`

## How to generate:
See `PLAN_patient_encounter_v2.md` → Section 6 (Ready Player Me GLB Generation Workflow).

1. Go to readyplayer.me
2. Create avatar with appropriate demographic (see demographic guide in plan)
3. Download as GLB with Draco compression + morph targets enabled
4. Rename and place here

## File size targets:
- < 3 MB each (Draco compressed)
- 1K or 2K textures
- 52 ARKit blend shapes required (check "include morph targets" on export)
